## Инструкции по установке
1. Установить VS Code
2. С помощью команды `git clone https://github.com/Tasmahat/mobile_lab` скачать исходный код
3. Зайти в папку с установленным проектом
4. Открыть в папке командную строку
5. Запустить команду `npm install`
6. Запустить команду `npm start`

## Краткое описание принятых решений при реализации
В качестве главной страницы выступает карта, приближенная на Пермь. Для обработки ошибок при выведении карты используется BoundaryError. 

При долгом нажатии в состояние записываются координаты нажатия. Они в свою очередь используются в классе MarkerList для выведения на экран всех маркеров, координаты которых пользователь ввел с помощью долгого нажатия.

По нажатии на маркер, пользователь с помощью router.navigate переходит на страницу маркера. Пользователь может добавить картинку. При долгом нажатии на картинку всплывает алерт, уточняющий, точно ли пользователь хочет удалить изображение.

Для обработки ошибки при добавлении изображения используется try{}catch(){}.
